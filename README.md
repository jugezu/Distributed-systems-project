# Distributed-systems-project


### Do the following to run the system:
<br/>
<br/>


1. Go to the correct folder: ```cd .\Distributed-systems-project\```
##
2. Create virtual environment: ```python -m venv venv```
##
3. Start the environment: ```venv\Scripts\activate```
##
4. Install dependencies: 
    ```pip install -r requirements.txt```

   Or
   
    ```pip install requests```
   
    ```pip install bcrypt```
   
    ```pip install flask```
   
    ```pip install pymongo```
   
    ```pip install PyJWT```
   
    ```pip install colorama```
    
   ### Requirements.txt contains packages that come with other dependencies that are not used in this project.


##
5. open multiple terminals and run each file:

    ```python forum.py```
   
    ```python apiGateway.py```
   
    ```python authentication.py```
   
    ```python client.py```
