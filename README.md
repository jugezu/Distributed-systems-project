# Distributed-systems-project


#Run the program 

1. cd .\Distributed-systems-project\

2. venv\Scripts\activate

3. Install dependencies: 
    pip install -r requirements.txt 

    Requirements.txt contains packages that come with other dependencies that are not used in this project.

    or : 

    pip install requests
    pip install bcrypt
    pip install flask
    pip install pymongo
    pip install PyJWT
    pip install colorama

4. open multiple terminals and run each file:

    python forum.py 
    python apiGateway.py
    python authentication.py
    python client.py
